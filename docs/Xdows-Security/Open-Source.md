# 开源说明
## Xdows Security 本身
Xdows Security 本身的源代码是基于 MIT 协议开源的，但其资源（图片等）、脚本等需要经过允许才能使用。

如果使用了该开源项目，我们希望您联系我们以加入到 [代码授权][Code] 页面

## Xdows Securtiy 使用的项目

Xdows Securtiy 使用的项目都会添加到关于页面

### 使用的项目进行开源

如果您允许开源，我们将把使用您项目的源代码也进行开源，如果您不允许，我们也会通过包装DLL等方式不进行开源
