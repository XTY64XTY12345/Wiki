# 版本计划

## N 版本

### 简介

Xdows Security N 是 Xdows Security 的轻量版。

### 区别

Xdows Security N 对比标准版会有如下改动。

 - 使用原生组件
 - 缩减扫描功能
 - 降低软件占用
 - 仅必要的 Xdows Tools 组件

### 获取

Xdows Security N 还没有发布，关注 QQ 群以获取相关进展。

## Pro 版本

### 简介

Xdows Security Pro 是 Xdows Security 的增强版。

### 区别

Xdows Security Pro 对比标准版会有如下改动。

 - 更多 Xdows Tools
 - 不开源
 - 支持免费定制
 - 勒索病毒防护
 - 加密文件恢复（依靠备份）
 - 驱动级进程
 - 系统奔溃（蓝屏）数据收集
